Dicionário de Tags HTML

Lista de Tags Utilizadas

<html>
Define o documento como um documento HTML.
<head>
Contém metadados/informações para o documento.
<title>
Define o título do documento, mostrado na barra de título do navegador ou na guia da página.
<meta>
Define metadados que não podem ser definidos usando outro elemento HTML.
<link>
Usado para vincular a folhas de estilo externas, entre outras coisas.
<body>
Define o corpo do documento e contém todo o conteúdo da página, como texto, hiperlinks, imagens, tabelas e listas.
<header>
Define um cabeçalho para um documento ou uma seção.
<footer>
Define um rodapé para um documento ou seção.
<main>
Define o conteúdo principal do documento.
<section>
Define uma seção em um documento.
<ul>
Define uma lista não ordenada.
<li>
Define um item da lista dentro de uma lista ordenada (<ol>) ou não ordenada (<ul>).
<a>
Define um hiperlink, que é usado para vincular de uma página a outra, ou a uma seção dentro da mesma página.
<address>
Define informações de contato para o autor/proprietário de um documento ou artigo.
<img>
Define uma imagem.
<h1> a <h6>
Define cabeçalhos de diferentes tamanhos.
<p>
Define um parágrafo.
<div>
Define uma divisão ou seção em um documento.
<span>
Define uma parte de um texto que deve ser estilizada de forma diferente de seu conteúdo circundante.
<code>
Define um trecho de código.
<strong>
Define um texto importante.
<dt>
Define um termo em uma lista de descrição.
<dd>
Define uma descrição para o termo anterior (<dt>).
<dl>
Define uma lista de descrição.
